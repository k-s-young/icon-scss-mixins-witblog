# WIT Icon Mixins(SCSS)

Collection of SCSS icon mixins.
See [blog article](http://wit.nts-corp.com/2017/10/31/4972)

## INSTALLATION

You can install it via npm:
```
npm install --save-dev icon-scss-mixins-witblog
```
